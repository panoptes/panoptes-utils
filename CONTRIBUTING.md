# Contributing

We use a single-remote workflow. The primary branch is `main`.

1. **Clone the repository** (do not fork):
   ```bash
   git clone https://github.com/panoptes/panoptes-utils.git
   ```
2. **Create a feature branch** from `main`:
   ```bash
   git checkout -b feature/your-feature-name main
   ```
3. **Submit a Pull Request** against `main`.

See [the POCS wiki](https://github.com/panoptes/POCS/wiki/PANOPTES-Feature-Development-Process)
for more info on general PANOPTES contribution guidelines.
